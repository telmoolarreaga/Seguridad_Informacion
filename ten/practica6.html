Práctica 06

Medidas de seguridad de la información

BACKUP STRATEGY PYTHON AUTOMATION RAID 10
01

Diseño BackUp: Empresa (30 trabajadores)

Sistemas y Precio

Una empresa de 30 trabajadores manejaría un volumen grande de datos de cliente y código fuente. Por esta razón, consideramos que unas de las mejores opciones son, un servidor NAS o Network Attached Storage, en RAID 10 para copias locales rápidas. Podríamos combinar el sistema con un servicio de Cloud Backup empresarial como por ejemplo AWS S3 Glacier o Azure Archive.

Coste: 1200 a 2000 euros. Almacenamiento cloud aprox 0,02 €/GB al mes.

Alternativa Económica

La alternativa más barata y segura sería Sustituir el NAS por un servidor dedicado antiguo reutilizado con discos nuevos y usar Backblaze B2 como nube, que suele tener costes de salida menores.

02

Diseño BackUp: Autónomo (1 trabajador)

Sistemas y Precio

En este segundo caso, al ser un único trabajador consideramos que es más importante priorizar la agilidad y que el coste sea pequeño. Nos decantamos por usar un Disco Duro Externo SSD que realice por ejemplo copias diarias rápidas. Esto lo combinaremos con algún servicio de Cloud Sync por ejemplo Google Drive Business y así poder tener un historial de versiones.

Coste: SSD (150-200€) + Cloud (15€/mes).

Alternativa Económica

Eliminar el disco externo físico y optar por soluciones 100% en nube gratuitas o de muy bajo coste:

  • GitHub/GitLab gratuito (repositorios privados).
  • Google Drive / OneDrive 100 GB (1,99 €/mes).
  • Scripts de mysqldump en el propio VPS (~0 €/mes).
03

Automatización con Python

backup_system.py 
import os
import shutil
import time
from datetime import datetime

ORIGEN = "disco_a"
DESTINO = "disco_b"
LOG = "backup_log.txt"
INTERVALO = 5  # segundos

def escribir_log(nivel, mensaje):
    """Mejora los logs incluyendo el nivel de importancia y más detalle """
    timestamp = datetime.now().strftime('%Y-%m-%d %H:%M:%S')
    with open(LOG, "a", encoding="utf-8") as f:
        f.write(f"[{timestamp}] [{nivel}] {mensaje}\n")

def copiar_nuevos_y_modificados():
    for carpeta_actual, subcarpetas, archivos in os.walk(ORIGEN):
        ruta_relativa = os.path.relpath(carpeta_actual, ORIGEN)
        carpeta_destino = os.path.join(DESTINO, ruta_relativa)

        if not os.path.exists(carpeta_destino):
            os.makedirs(carpeta_destino)
            escribir_log("INFO", f"Nueva estructura de directorios creada: {carpeta_destino}")

        for archivo in archivos:
            ruta_origen = os.path.join(carpeta_actual, archivo)
            ruta_destino = os.path.join(carpeta_destino, archivo)

            if not os.path.exists(ruta_destino):
                shutil.copy2(ruta_origen, ruta_destino)
                escribir_log("NUEVO", f"Archivo copiado por primera vez: {archivo} en {carpeta_destino}")
            
            else:
                fecha_origen = os.path.getmtime(ruta_origen)
                fecha_destino = os.path.getmtime(ruta_destino)

                if fecha_origen > fecha_destino:
                    nombre, extension = os.path.splitext(archivo)
                    fecha_str = datetime.now().strftime("%Y-%m-%d_%H-%M-%S")
                    nombre_versionado = f"{nombre}_{fecha_str}{extension}"
                    ruta_versionado = os.path.join(carpeta_destino, nombre_versionado)

                    shutil.copy2(ruta_destino, ruta_versionado)
                    shutil.copy2(ruta_origen, ruta_destino)
                    
                    tamano = os.path.getsize(ruta_destino)
                    escribir_log("MODIFICADO", 
                                 f"Archivo '{archivo}' actualizado ({tamano} bytes). "
                                 f"Versión previa guardada como: {nombre_versionado}")

def main():
    if not os.path.exists(ORIGEN): os.makedirs(ORIGEN)
    if not os.path.exists(DESTINO): os.makedirs(DESTINO)

    print(">>> Sistema de Backup con Versionado Activo <<<")
    escribir_log("SISTEMA", "Inicio del servicio de monitorización y backup.")

    try:
        while True:
            copiar_nuevos_y_modificados()
            time.sleep(INTERVALO)
    except KeyboardInterrupt:
        escribir_log("SISTEMA", "Servicio detenido por el usuario.")
        print("\nSistema detenido.")

if __name__ == "__main__":
    main()

Valoración de la práctica

Utilidad

Utilidad muy alta. Entender la diferencia entre backups locales y en la nube es vital para la continuidad de cualquier negocio.

Dificultad

Dificultad media Nos ha parecido un poco más dificil que las otras practicas por la segunda parte de la práctica en la que ha habido que programar.

¿Qué hemos aprendido?

Hemos aprendido a poner en práctica la teoría que hemos visto en clase este Martes. También hemos aprendido a diseñar planes de contingencia escalables según el tamaño de la empresa. Además, hemos automatizado el versionado de archivos, una técnica muy importante para eliminar daños por errores humanos o ataques de integridad.
